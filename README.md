# Personal Linktree

Built a modern Linktree page with React, Vite and CSS. Links include Instagram, LinkedIn, GitHub, and my personal Resume. 
Includes Profile picture and a small bio, as well as, Javascript features to make the page feel more interactive


## Features

- Profile Section
- Links to LinkedIn, GitHub, Resume, and Instagram
- Custom CSS layout
- Javascript feature: time-based greeting and day/night mode + toggle button

## Javascript feature
I created a time based greeting and day/night mode toggle as my feature. I chose this because I initially realized my color palette was a bit too bright when viewing in darker areas/times. So to help ease the eyes, I created a Night mode feature which should automatically turn on when the time hits 6PM to 6AM. I also created a greeting along with this to make it feel more interactive.
